# COVID-19 Data Analysis Project

## Project Overview
This project analyzes global COVID-19 data to understand trends in confirmed cases, deaths, recoveries, and testing statistics using Python and visualization libraries.

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Plotly
- Jupyter Notebook

## Dataset
The project uses COVID-19 datasets containing country-wise statistics such as:
- Total Cases
- Total Deaths
- Total Tests
- Recoveries
- Continent-wise information

## Project Workflow
1. Imported and explored datasets.
2. Performed data cleaning and preprocessing.
3. Analyzed country-wise and continent-wise COVID statistics.
4. Created interactive visualizations using Plotly.
5. Derived insights from the analysis.

## Visualizations Created
- Country-wise Total Cases Analysis
- Country-wise Total Deaths Analysis
- Total Tests Analysis
- Continent-wise Scatter Plots
- Interactive Bar Charts

## Key Insights
- Identified countries with the highest COVID-19 cases.
- Compared testing statistics across different countries.
- Analyzed continent-wise distribution of cases and tests.
